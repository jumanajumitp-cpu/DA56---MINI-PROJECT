# Fashion Boutique Sales Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed to analyze fashion boutique sales and business performance.

## Tools Used
- Power BI
- Microsoft Excel

## Features
- Sales Analysis
- Product Performance
- Category-wise Sales
- Interactive Dashboard
- Business Insights

## Files Included
- MINI PROJECT (Power BI).pbix
- fashion_boutique_dataset.csv
